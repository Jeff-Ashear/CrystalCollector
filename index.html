Match the number on the top right. This is achieved by clicking on the crystal skulls. Each skull represents a unique random number between 1 and 10. Each time you click a skull it's number will be added to your score. If your score exceeds the "Match Me" number you lose the round. If you match the number, you win that round! Each round your score is set to zero and the number to match as well as each skull's number is re-rolled. Have fun!
